
# Risk-Based Monitoring (RBM) Dashboard | Power BI & SQL Server

## Overview
Developed an interactive Risk-Based Monitoring dashboard for a simulated multicenter Phase III clinical trial. The dashboard helps identify high-risk sites, monitor patient safety metrics, and support monitoring decisions using SQL Server and Power BI.

(Your dashboard image)

## Project Objectives
- Monitor site performance
- Identify high-risk sites
- Track Serious Adverse Events (SAEs)
- Analyze query resolution time
- Recommend monitoring strategy

## Key Features
✔ KPI Cards
✔ Risk Distribution (Donut Chart)
✔ Top 5 High Risk Sites
✔ Monitoring Recommendation
✔ Interactive Filters
✔ Site Summary Table
✔ Key Insights

## Tools & Technologies
- Power BI
- SQL Server
- Power Query
- DAX
- Microsoft Excel

## KPIs Used
- Total Sites
- Total Patients
- Total SAEs
- High Risk Sites
- Open Queries
- Average Query Resolution Time

## Insights
- 1 site identified as High Risk.
- 9 sites classified as Low Risk.
- Average query resolution time: 6.29 days.
- Remote monitoring recommended for most sites.
