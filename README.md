# Risk-Based Monitoring (RBM) & Safety Analytics Dashboard

## Project Overview

This project demonstrates an end-to-end Risk-Based Monitoring (RBM) dashboard developed using Microsoft SQL Server and Microsoft Power BI for a simulated multicenter Phase III Oncology Clinical Trial.

The dashboard helps identify high-risk clinical trial sites, monitor safety events, analyze open queries, and recommend appropriate monitoring strategies.

---

## Tools Used

- Microsoft SQL Server
- Microsoft Power BI
- Microsoft Excel

---

## Dashboard Features

### KPI Cards
- Total Sites
- Total Patients
- Total Serious Adverse Events (SAEs)
- Open Queries
- High Risk Sites
- Average Query Resolution Time

### Visualizations
- Risk Distribution (Donut Chart)
- Top 5 High Risk Sites
- Monitoring Recommendation
- Site Summary Table
- Interactive Slicers
- Conditional Formatting
- Key Insights
